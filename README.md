# Create a Web App for Machine Learning model with MLJAR AutoML

This is code repository for tutorial how to create a **Web App** for Machine Learning model trained with [MLJAR AutoML](https://github.com/mljar/mljar-supervised). 

We will train and deploy Web App for predicting patient hospitalization charges based on demographic and health risk metrics.

There are many files and directories in this repo, but the main script is `train_automl.py` - it is doing all the work. It has 26 lines of code, most are empty lines and comments :)

Full article: [How to create a Web App for Machine Learning model](https://mljar.com/blog/web-app-machine-learning/).

## What will you learn in this tutorial

1. We will train Machine Learning model to predict patient hospitalization charges. Training is done with Automated Machine Learning python pacakge `mljar-supervised`.
2. We will generate Web App to interact with ML model through GUI, we will use `mercury` framework.
3. We will deploy the Web App to the internet.


## Data

The data files are in `data` directory. It is a patient hospitalization charges dataset. The data is manually splited into `train.csv` and `test.csv`. The full dataset can be found in [pplonski/datasets-for-start](https://github.com/pplonski/datasets-for-start/tree/master/insurance). 

Columns in dataset:

- `age` - age of the person
- `sex` - gender
- `bmi` - body mass index
- `children` - number of children/dependents
- `smoker` - smoking status
- `region` - residential region
- `charges` - medical insurance charges

This is a regression task where we need to predict medical insurance charges based on demographic and health risk factors.

## Machine Learning 

For training we user MLJAR AutoML because it is simple to use and produce great results - please check `train_automl.py`. The report from training is available in this repository, in directory [AutoML_Results](https://github.com/pplonski/web-app-from-machine-learning-model/tree/main/AutoML_Results). 


Training code:

```python
# Train AutoML
automl = AutoML(
    results_path="AutoML_Results",
)
automl.fit(X_train, y_train)
```

## Web App

Web App is generated by MLJAR AutoML in automated way. You dont need to code anything. Code to generate app after training:

```python
# Create App
automl.app(
    path="AutoML_App",
    title="Insurance Charges Predictor",
)
```

Web App is created with [Mercury framework](https://github.com/mljar/mercury). The Web App code is available in directory [AutoML_App](https://github.com/pplonski/web-app-from-machine-learning-model/tree/main/AutoML_App).

The Web App have to apps:
- application to compute prediction on single sample,
- application to compute batch predictions on CSV file.

## Deployment

You can start Web App locally with `mercury` command. There is also a Python function that can start a local server directly after training:

```python
# Start App locally
automl.local_app()
```

Web App can be published online with Docker - please check [instructions how to create Dockerfile](https://runmercury.com/deploy/dockerfile/).

The easiest option is to deploy Web App with [MLJAR Platform](https://platform.mljar.com) with function:

```python
# Deploy App on platform.mljar.com
automl.publish_app()
```

It will automatically upload your files to [platform.mljar.com](https://platform.mljar.com) and create an URL for your application.
